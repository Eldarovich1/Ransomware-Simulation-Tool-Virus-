📋 What Is This?
This is a ransomware simulation tool created for cybersecurity education. It mimics real ransomware behavior to help people understand how these threats work in a safe environment.

This is NOT real malware. This is a SAFE educational simulation.

The program file is named Microsoft Defender.exe to demonstrate how attackers disguise their software as legitimate security tools.
